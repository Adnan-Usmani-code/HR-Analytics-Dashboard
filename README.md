# 📊 HR Analytics Dashboard – Power BI

## 📌 Project Overview
This Power BI project analyzes HR data to understand employee attrition trends, salary distributions, job roles, and workforce demographics. It visualizes key metrics to help HR professionals make data-driven decisions.

---

## 🛠 Tools Used
- **Power BI Desktop**
- DAX
- Data Modeling

---

## 📈 Key Metrics
- **Total Employees:** 1470  
- **Attrition Count:** 237  
- **Attrition Rate:** 16.1%  
- **Average Age:** 37  
- **Average Salary:** 6.5K  
- **Average Years at Company:** 7
